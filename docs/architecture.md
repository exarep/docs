# Architecture

An overview of the Exarep system architecture.

## Components

Details on system components will be added as the project develops.

## Technology stack

| Layer         | Technology                  |
| ---           | ---                         |
| Runtime       | Quarkus                     |
| Language      | Java (latest LTS)           |
| Frontend      | Angular with ng-bootstrap   |
| Database      | PostgreSQL                  |
| Container     | Podman                      |
| Orchestration | OpenShift                   |
| GitOps        | OpenShift GitOps (Argo CD)  |
| CI/CD         | OpenShift Pipelines         |
| Auth          | OIDC (Keycloak reference)   |
