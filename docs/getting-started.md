# Getting started

Set up your development environment and run Exarep locally.

## Prerequisites

- RHEL/Fedora or compatible Linux distribution
- Podman
- Java (latest LTS)
- Maven
- Node.js (latest LTS)

## Installation

```shell
git clone https://github.com/exarep/exarep.git
cd exarep
```
